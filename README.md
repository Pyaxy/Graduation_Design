# 基于Django的程序设计课程辅助管理平台

## 🚀项目介绍
- 构建一个面向学生与教师的 **协作编程教学平台**，支持：
    - 用户功能：
        - 学生与教师通过 **账号密码** 登录
        - 学生与教师通过 **账号密码** 注册
        - 学生与教师通过 **账号密码** 退出登录
    - 教师功能：
        - 教师可以创建课程(CRUD)
        - 教师可以创建课题(CRUD)
    - 学生功能：
        - 学生可以查看课程(JOIN and LEFT)
        - 学生可以查看课题


## 🚀如何部署
- 由于本项目使用Docker进行了容器化，所以需要先安装Docker。
- 安装完成后，进入到项目根目录，使用Docker Compose进行部署。
```
cd /path/to/project
docker-compose up -d
```
- 部署完成后，访问http://localhost:5173，即可看到项目的首页。
